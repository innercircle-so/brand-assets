# Inner Circle Brand Assets

**Official brand assets for Inner Circle - India's premier AI community**

## 📁 What's Inside

### 🎨 Logos & Wordmarks (`/logos/`)
- **Main Logo**: Black and white versions (PNG/SVG)
- **Wordmark**: Black and white background variations
- **File formats**: PNG for web/print, SVG for scalable use

### 🔤 Font System (`/fonts/`)
- **Primary**: Space Grotesk (display, headers)
- **Secondary**: Satoshi Complete (body text)
- **Accent**: NulTien (special cases)
- **Formats**: Complete font families with all weights

### ⚡ Graphic Elements (`/graphics/`)
- **Design Assets**: Core visual elements and patterns
- **SVG Components**: Scalable graphic components
- **Supporting Elements**: Accent graphics and icons

### 🎓 Academy Sub-Brand (`/academy/`)
- **Academy Logo**: Dedicated branding for Inner Circle Academy
- **Variations**: Black and white versions (PNG/SVG)
- **Use**: For educational content and workshops

## 🎯 Usage Guidelines

### Logo Usage
- **Minimum size**: 24px height for digital, 0.5" for print
- **Clear space**: Logo height on all sides
- **Backgrounds**: Use appropriate contrast versions

### Typography
- **Headers**: Space Grotesk (Bold/Medium)
- **Body text**: Satoshi (Regular/Medium)
- **Special**: NulTien for distinctive elements

### Colors
- **Primary**: Black (#000000)
- **Secondary**: White (#FFFFFF)
- **Accent**: Match your content/platform aesthetic

## 🚀 Quick Start

1. **Download what you need** from the appropriate folder
2. **Check file format**: SVG for scalable, PNG for fixed use
3. **Maintain proportions**: Don't stretch or distort logos
4. **Use proper contrast**: Black on light, white on dark
